## How to configure
Instructions for setting up project
- Clone repository to the local system
    ```bash
    git clone https://github.com/Shakmoot/SHN_Code_Crocodilezzz.git 
    ```
- venv
    ```bash
    python -m venv .venv 
    ```
- activate it
    ```bash
    .\.venv\Scripts\Activate.ps1
    ```

## How to Run
- To run the project
    ```bash
    flet run main.py
    ```
